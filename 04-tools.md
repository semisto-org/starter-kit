# 4. Outils & Stack numérique

Nos outils servent la transparence, la collaboration distribuée et l’automatisation. Voici les fondamentaux pour tout membre Semisto.

## 4.1 Terranova (plateforme réseau)
- **Ce que c’est** : CRM + gestion de projets + Academy + Roots + Nursery.
- **Accès** : compte Google → groupe `terranova-beta`. Voir guide détaillé dans Compass (section Terranova) et l’onboarding bêta.
- **Modules actifs (2026)** :
  - Design Studio : pipeline, étapes OBREDIM, documents.
  - Academy : catalogue, sessions, attendance, mails.
  - Roots : base bénévoles, événements, reporting.
  - Nursery : stocks (lecture), fiches espèces.
- **À venir** : inventaires complets, tracking financier, exports comptables.
- **Bonnes pratiques** :
  - Logguez vos interactions client/bénévole le jour même.
  - Utilisez les tags (`#beta-feedback`, `#urgent`) pour repérer les besoins produit.
  - Toute décision liée à un projet doit avoir une note Terranova associée.

## 4.2 Slack
- **Usage** : communication synchrone, coordination quotidienne.
- **Canaux essentiels** :
  - `#annonces` (communication officielle) – lecture obligatoire.
  - `#aujourdhui` – check-ins journaliers, demandes rapides.
  - `#life` – culture, entraide, annonces perso.
  - Pôles (`#design`, `#academy`, `#nursery`, `#roots`) et cercles.
- **Règles clés** :
  - Parlez au **tu**, ton chaleureux (cf. Manuel Communication).
  - Documentez les décisions hors Slack (Compass/Terranova) et partagez le lien.
  - Utilisez les threads pour garder les discussions lisibles.

## 4.3 Notion & Compass
- **Compass** (GitHub + Astro) : documentation publique/partagée, playbooks, politiques.
- **Notion** : espaces de travail temporaires, notes de réunions, suivi projets internes (quand Terranova ne suffit pas encore).
- **Règle** : quand le processus se stabilise, migrez-le vers Compass ou Terranova.

## 4.4 Stack IA & automatisations
- **ChatGPT / Codex / Nova** : assistance rédaction, synthèse, prototypage. Toujours relire et “humaniser” avant diffusion (cf. skill humanizer).
- **Automations** :
  - Scripts Linkinator (sites web) pour vérifier les liens.
  - Zapier/Make pour newsletters, formulaires bénévoles → Terranova.
  - Jobs cron (veille stratégique, daily briefs) documentés dans Compass.

## 4.5 Outils terrain
- Drive partagé (plans, photos, scans de notes).
- QGIS / Illustrator pour cartographie.
- Sheets “Nursery” pour suivi des semis (en attendant le module Terranova complet).

## 4.6 Sécurité & accès
- Comptes Google gérés via l’ASBL Semisto (2FA obligatoire).
- Gestion des secrets (API, tokens) via le dépôt `secrets/` sécurisé.
- Toute nouvelle automation doit être consignée dans Compass + revue par Bee (Super Génial).

> 🔁 Pensez à alimenter Compass / Starter Kit à chaque fois qu’un outil ou processus change. Un bon outil est celui que tout le monde comprend et documente.
