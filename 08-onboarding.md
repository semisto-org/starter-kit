# 8. Onboarding & Vie pratique

Parcours recommandé pour tout nouveau membre (salarié·e, freelance, bénévole longue durée).

## 8.1 Checklist 48 h
1. **Accès**
   - Compte Google Semisto (SSO).
   - Slack + canaux essentiels (`#general`, `#aujourdhui`, pôle).
   - Terranova (si rôle opérationnel) → suivre le guide Onboarding Terranova.
   - GitHub (Compass/Starter Kit) si contribution docs.
2. **Docs à lire**
   - Ce Starter Kit (`README` + sections pertinentes).
   - Compass > Playbook (sections correspondant à votre pôle).
   - Guide Slack/Notion si besoin.
3. **Rencontres**
   - 1:1 avec votre sponsor (personne qui vous parraine).
   - Présentation rapide durant le prochain stand-up.

## 8.2 Semaine 1
| Jour | Actions |
| --- | --- |
| J1 | Tour d’horizon Terranova + Compass, config Slack + pronoms |
| J2 | Observation terrain / design (selon pôle) |
| J3 | Participer à un rituel (stand-up, demo) |
| J4 | Prendre en charge une mini-tâche documentée |
| J5 | 1:1 feedback avec sponsor |

## 8.3 Semaine 4
- Être autonome sur les outils principaux.
- Avoir documenté au moins une ressource (note Terranova, page Compass, retour Slack).
- Participer à un Semisto Day ou un chantier Roots si possible.

## 8.4 FAQ express
**Q : Dois-je maîtriser tous les outils dès le début ?**
> Non. Commencez par Terranova (si rôle opérationnel) + Slack. Notion/Compass viennent ensuite.

**Q : Comment demander de l’aide ?**
> Ping ton sponsor ou poste dans le canal pôle avec le tag `[QUESTION]`. Les Semisti répondent vite quand la question est claire.

**Q : Où signaler un bug / une idée ?**
> Slack `#terranova-beta` ou formulaire dans Compass > Terranova.

**Q : Comment proposer une amélioration process ?**
> Documente la tension (Contexte → Besoin → Proposition) puis soumets-la lors du prochain stand-up ou via une note Compass.

## 8.5 Vie pratique
- **Horaires** : flexibles, coordonnés par pôles. Les chantiers terrain nécessitent une présence physique.
- **Matériel** : chaque Lab gère son stock. Pour les missions terrain : vêtements adaptés, chaussures sécurité, gants.
- **Remu & contrats** : Playbook financier (Compass) + contact Admin.
- **Santé & sécurité** : fiches signalétiques sur Compass + brief obligatoire avant chaque chantier.
