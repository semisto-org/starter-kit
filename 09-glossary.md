# 9. Glossaire & Ressources

| Terme | Signification |
| --- | --- |
| **Terranova** | Plateforme interne : CRM, gestion projets, Academy, Roots, Nursery. |
| **Compass** | Wiki/knowledge base (Astro + GitHub) contenant le Manuel de Semisto. |
| **Starter Kit** | Cette documentation GitBook, orientée onboarding rapide. |
| **OBREDIM** | Méthode de design : Observation, Bordures, Ressources, Évaluation, Design, Implémentation, Maintenance. |
| **Lab** | Antenne locale Semisto (ex : Lab Wallonie, Lab Île-de-France). |
| **Roots** | Pôle mobilisation citoyenne (Food Forest Heroes). |
| **Semisto Days** | Journées de rencontre/co-création de l’équipe. |
| **Cycle 7+1** | Cadence opérationnelle : 7 semaines de production + 1 semaine de respiration. |
| **Semos** | Monnaie interne pour valoriser les contributions (cf. Playbook financier). |
| **Bee / Super Génial** | Equipe technique responsable des outils (Terranova, infra). |
| **OBM** | Observations Botanique & Maintenance (follow-up plantation). |
| **NPS** | Net Promoter Score, indicateur de satisfaction. |

## Ressources externes inspirantes
- [Braver Help Center](https://support-en.braver.net) – exemple de documentation claire et orientée utilisateur.
- [Basecamp Handbook](https://basecamp.com/handbook) – culture d’équipe transparente.
- [LIFE Programme EU](https://cinea.ec.europa.eu/programmes/life_en) – financement projets environnementaux.

## Comment contribuer
1. Créez une branche ou un fork du dépôt `semisto-org/starter-kit`.
2. Ajoutez/éditez les fichiers Markdown concernés.
3. Soumettez une PR en expliquant la source (Manuel, décision Slack, notes terrain…).
4. Notifiez le/la Gardien·ne du Manuel pour revue.

Merci de faire vivre cette documentation au rythme de Semisto ! 🌿
