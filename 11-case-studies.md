# 11. Cas pratiques

Pour aider les nouveaux Labs à se projeter, voici des mini-cas inspirés de situations réelles (simplifiés et anonymisés).

## Cas 1 – Design : Forêt-jardin périurbaine (Lab Wallonie)
- **Contexte** : commune de 25 000 habitants souhaitant convertir 2 ha en espace nourricier.
- **Approche** :
  1. Diagnostic OBREDIM (topographie, flux habitants, sol argilo-limoneux).
  2. Atelier participatif avec riverains (mapping besoins, espèces préférées).
  3. Design en 4 strates + zone pédagogique.
  4. Budget estimatif + planning de chantier participatif.
- **Outcome** : financement mix municipal + mécénat, chantier Roots en 3 vagues, monitoring biodiversité via Terranova.

## Cas 2 – Academy : Parcours “Design Forest Gardens” (Lab Île-de-France)
- **Contexte** : version francilienne d’un PDC, 60h hybrides.
- **Clés de succès** :
  - Modules asynchrones hébergés sur Terranova, accompagnement Slack dédié.
  - Sessions terrain dans deux sites pilotes.
  - Co-animation avec alumni (mentorats).
- **Indicateurs** : 24 apprenants, 92% de complétion, 5 projets concrets lancés dans les 6 mois.

## Cas 3 – Nursery : Production de plants climat-méditerranéens (Lab Catalogne)
- **Contexte** : besoin d’espèces résilientes sécheresse (caroubiers, pistachiers, aromatiques).
- **Process** :
  - Plan de multiplication sur 18 mois (semis + boutures).
  - Suivi des lots dans Terranova (statut, conditions de culture).
  - Partenariat avec une école horticole pour stages.
- **Result** : 1 800 plants livrés, 85% taux de reprise, documentation partagée avec les autres Labs.

## Cas 4 – Roots : Campagne “1000 arbres en musique” (Lab Rheineland)
- **Contexte** : mobilisation grand public, reboisement convivial.
- **Actions** :
  - Événement Roots avec concerts + ateliers.
  - Sponsors locaux pour financer les plants.
  - Terranova utilisé pour inscriptions, tâches bénévoles, collecte feedback.
- **Leçons** : importance d’une équipe logistique dédiée, docs ready-to-use (brief sécurité, checklists).

## Cas 5 – Partenariats & financement : appel LIFE “Urban Greening”
- **Contexte** : candidature multi-Labs (Wallonie + Catalogne) pour un projet européen.
- **Étapes** :
  1. Constitution consortium (villes, universités, ONG).
  2. Collecte des données impact (Terranova) pour le dossier.
  3. Rédaction conjointe via GitHub/Notion → export PDF.
  4. Préparation plan de gouvernance (cercle projet).
- **Résultat** : pré-sélection + financement partiel, extension prévue en 2027.
